# DDIT-final
대덕인재개발원 최종프로젝트

# 프로젝트 개요
그룹웨어 및 사용자 페이지 관리 기능을 제공하는 서비스

# 주요 기능
 - 직원페이지 로그인/로그아웃
 - Tiles를 이용한 권한 별 메뉴 설정
 - WebSocket을 이용한 실시간 채팅
 - PdfBox, itextPdf를 이용한 급여명세서/출입증 신청서 관리
 - 토스페이먼츠를 이용한 주차예약
 - 한국공항공사 API를 활용한 실시간 탑승 수속 소요시간, 항공편 조회
